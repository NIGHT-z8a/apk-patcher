Features

📦

Bundle Support

Extracts .apks bundles, merges split APK native libraries into a single installable APK.

🔧

Smali Injection

Automatically injects mod menu hooks into the game's main activity onCreate method.

🔒

APK Signing

Signs with apksigner v2/v3 scheme, preserving zipalign alignment for Android 11+ compatibility.

📐

Zipalign

Properly aligns uncompressed files (.arsc, .so) to 4-byte boundaries for optimal performance.

⚡

CLI Arguments

Full control via command line: input, output, architecture, library name, SDK path, and more.

🎯

Auto SDK Detection

Finds apksigner via ANDROID_HOME, ANDROID_SDK_ROOT, or common system paths automatically.

Quick Start

Terminal 
# Clone the repository
git clone https://github.com/NIGHT-z8a/apk-patcher.git
cd apk-patcher

# Install dependencies
bash setup.sh

# Run with a regular APK
python3 main.py --input game.apk --modmenu app-debug.apk

# Run with a split APK bundle
python3 main.py --input game.apks --modmenu app-debug.apk

# With custom options
python3 main.py -i game.apks -m menu.apk -o ./output -a armeabi-v7a
